🌍 Country Explorer
📌 Description

Country Explorer is a web application that allows users to explore information about countries around the world. Users can search for countries, filter them based on region and population, and sort them according to different criteria.

🎯 Objective

The goal of this project is to demonstrate the use of JavaScript, API integration, and UI development by building an interactive web application.

🚀 Features
Search countries by name
Filter countries by region (Asia, Europe, etc.)
Filter countries based on population
Sort countries by name or population
Display country details such as flag, capital, region, and population
🔗 API Used

REST Countries API
https://restcountries.com/

🛠️ Technologies Used
HTML
CSS
JavaScript (Fetch API)
(Optional: Tailwind CSS / Bootstrap)
📁 Project Structure
index.html → Structure of the web page
style.css → Styling of the application
script.js → JavaScript logic and API calls
⚙️ Setup Instructions
Clone the repository
Open the project folder
